# FitX Mobile App — UI Kit

A pixel-faithful recreation of the FitX home fitness & nutrition app. Built with React + JSX, no build step.

## What's here

- `index.html` — interactive click-thru of the core flows. Bottom-tab nav across **Home → Workouts → Meals → Coach → You**.
- `Frame.jsx` — iOS device frame wrapper.
- `Components.jsx` — buttons, badges, cards, headers, list items, bottom-nav.
- `Screens.jsx` — full screens (Home, Workout detail, Meal plan, Coach chat, Profile).

## Design notes

- All copy in FitX voice (warm, age-inclusive, no hype).
- Color tokens from `../../colors_and_type.css`.
- Lucide icons via inline SVG (so the kit works offline-ish).
- Photography is faked with warm placeholder gradients + initials. Replace with real shoots before launch.
